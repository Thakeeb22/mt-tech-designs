IoT-Based Student Attendance System

Designed and developed a smart attendance management system that uses Biometric fingerprint authentication to automate student attendance tracking. The system integrates an ESP8266 microcontroller, RFID scanner, fingerprint sensor, OLED/LCD displays, and a web-based dashboard for real-time attendance monitoring and management.
The solution eliminates manual attendance processes, reduces attendance fraud, and provides accurate attendance records through seamless hardware and software integration.

Frontend: HTML, CSS, JavaScript
Backend: PHP, MySQL
Hardware: ESP8266, Fingerprint Sensor, OLED Display, I2C LCD, Buzzer
